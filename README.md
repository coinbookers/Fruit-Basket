# Fruit-Basket
Fruit Basket , a beginner-friendly C++ console game designed to teach essential programming techniques through a playful collecting adventure. The project avoids external libraries and uses only standard C++ features, making it portable and easy to compile.

Players explore a virtual orchard where different fruits appear randomly during each round. Collecting an apple awards bonus points, while finding a banana provides a smaller reward but still contributes toward the final score. The simple mechanics create an engaging experience while keeping the code accessible to new programmers.

The implementation highlights common programming concepts such as loops, functions, conditional logic, and random event generation. The game structure differs from traditional guessing games by allowing the player to choose whether to continue exploring after each round, resulting in a different execution flow.

Developers can easily expand the project by introducing additional fruit varieties, inventories, weather conditions, or scoring multipliers. The clean organization and compact codebase make this repository an excellent foundation for learning how console games are built using only the C++ standard library.
